# JSS Security Suite

## Overview
JSS Security Suite is a set of practical Linux security and monitoring tools
designed for real server environments.

The project focuses on system visibility, basic hardening and automated
security reporting on Linux servers.

## Scope
- Linux server monitoring
- Log analysis
- Detection of suspicious activity
- Basic system security checks
- Automated security reporting

## Technology
- Linux (Ubuntu / Debian)
- Bash scripting
- systemd
- cron
- Standard Linux tools (ss, journalctl, ps, grep)

## Use case
Designed to be used on VPS or on-prem Linux servers as a lightweight
security and system status overview.

## Status
Actively developed as a hands-on Linux security and administration project.
